# IPPAX Frontend

Frontend do e-commerce IPPAX - Pisos Vinílicos.

## Arquitetura

- **Frontend**: React + Vite + TypeScript (Vercel)
- **Backend**: Node.js + Express (Coolify)

## Desenvolvimento Local

```bash
# Instalar dependências
npm install

# Rodar localmente
npm run dev

# Build para produção
npm run build
```

## URLs de Produção

- **Frontend**: https://www.ippaxfloor.com.br
- **Backend API**: [configurar URL do Coolify]

## Tecnologias

- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Query
- Wouter (routing)
